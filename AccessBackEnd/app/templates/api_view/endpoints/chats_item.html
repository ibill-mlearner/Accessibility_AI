• GET|PUT|PATCH|DELETE /api/v1/chats/<id>
    Details

    GET returns one chat by numeric id. Returns not_found if id does not exist.

    PUT/PATCH replace the full record with the provided JSON object.

    DELETE removes the record and returns the deleted object.
